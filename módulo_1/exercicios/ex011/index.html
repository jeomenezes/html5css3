Áudios no HTML5

Vamos aprender as duas formas de reproduzir áudios no HTML5.

Forma simplificada

Esta forma é limitada, não funciona em alguns navegadores e so carrega um arquivo de audio, se este não for comparivel com algum navegador o usuario deste fica na mão.

Forma completa

Esta forma usa a tag "audio" junto com a tag "source" para carregar dinamicamente varios formatos do mesmo arquivo de acordo com o navegador do usuario.
