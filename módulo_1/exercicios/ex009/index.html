Trabalhando com Listas

Listas Ordenadas

Uma lista ordenada é uma lista de itens que são numerados ou apresentados em uma sequência, usamos a tag "ol" para criar listas ordenadas e dentro dela usamos a tag "li" para criar cada item da lista.

  1. Acordar
  2. Fazer o numero 2
  3. Lavar o Rosto
  4. Matabichar
  5. Escovar os dentes

Listas não Ordenadas

Uma lista desordenada é uma coleção de itens que não têm uma ordem rígida. Cada item da lista é marcado da mesma forma, usamos a tag "ul" para criar listas não ordenadas e dentro dela usamos igualmente a tag filha "li" para criar os itens da lista.

  • Acordar
  • Lavar o Rosto
  • Ir comprar pão
  • Lavar a roupa
  • Escovar os dentes
  • Comer

Listas de definição

Uma lista de definição é uma lista que relaciona termos e suas definições. É um tipo de lista que pode ser usada para criar glossários ou catálogos de produtos.

Ulilizamos a tag "dl" para criar a lista, usamos a Tag "dt" para criar dentro da lista o termo que será definido, usamos a tag "dd" para definir o termo
