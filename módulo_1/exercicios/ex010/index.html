                Usando links internos: Resumo  
                
                href="link" rel="next" target="_self">  
            
                    - O link deve estar dentro da mesma pasta;
            
                    - Se o link estiver dentro de outra pasta deve ser informado o caminho completo.
            
                                  Ex:  href="noticias/pag003.html">  
            
                    - Ctrl + Espaço pode ajudar a selecionar o link;
            
                    -   ./    no link significa que o arquivo está na própria pasta;
            
                    -   ../   no link significa que o arquivo está na pasta anterior;
            
                        - Para voltar mais pastas em um link usa-se um ../ para cada pasta;
            
                                  Ex: "../../../arquivo.html"
            
                    - rel="next"  para link da página seguinte;
            
                    - rel="prev"  para link da página anterior;
            
                    - rel="nofollow"  informa ao navegador que você não dá aval à página;
            
                    - target="_self"  indica que o link será aberto na mesma guia (opcional);
